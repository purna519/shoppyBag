# Codebase Analysis Summary

## Project Overview
This is a full-stack e-commerce application named "ShoppyBag".

### Backend
- **Framework**: Spring Boot 3.5.6
- **Language**: Java 17
- **Build Tool**: Gradle
- **Database**: PostgreSQL (driver included)
- **Key Dependencies**:
    - Spring Data JPA (Database interaction)
    - Spring Security (Authentication/Authorization)
    - JJWT (JSON Web Tokens)
    - Razorpay (Payment Gateway)
    - SpringDoc OpenAPI (Swagger UI)
    - Lombok (Boilerplate reduction)
- **Structure**: Standard layered architecture (Controller, Service, Repository, Entity).

### Frontend
- **Location**: `shoppybag-frontend` directory
- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Bootstrap 5, React Bootstrap
- **HTTP Client**: Axios
- **Routing**: React Router DOM

## Directory Structure
- `src/main/java/com/shoppyBag`: Main backend source code.
    - `Authentication`, `Config`, `Controller`, `DTO`, `Entity`, `Exception`, `Filter`, `Integration`, `Model`, `Repository`, `Service`, `ServiceImpl`, `Util`.
- `shoppybag-frontend`: Frontend application source code.
